# E-Commerce Data Cleaning and Preparation

## Project Overview
This project focuses on cleaning and preparing an E-Commerce sales dataset using Python and Pandas. The dataset was processed by handling missing values, removing duplicate records, correcting data formats, and preparing the data for further analysis.

## Tools Used
- Python
- Pandas
- OpenPyXL
- Visual Studio Code

## Project Files
- Original Dataset
- Cleaned Dataset
- Python Source Code

## Data Cleaning Tasks
- Identified missing values
- Removed duplicate records
- Corrected data formats
- Cleaned text values
- Prepared the dataset for analysis

## Output
A cleaned dataset ready for Exploratory Data Analysis (EDA).
